# Vex Language Reference (A-Z)

A complete index of Vex language features, keywords, and concepts.

## A
- **Array**: Fixed-size sequences `[T; N]`. See [Data Types](./07_Structs_and_Data_Types.md).
- **Async/Await**: Concurrent programming. See [Concurrency](./13_Concurrency.md).
- **Auto-Vectorization**: Automatic SIMD/GPU optimization. See [Automatic Optimization](./25_Automatic_Optimization.md).

## B
- **Borrow Checker**: Ownership and safety system. See [Memory Management](./14_Memory_Management.md).
- **Box**: Heap allocation. See [Standard Library](./16_Standard_Library.md).

## C
- **Channel**: Message passing. See [Concurrency](./13_Concurrency.md).
- **Closure**: Anonymous functions `|x| x + 1`. See [Closures](./12_Closures_and_Lambda_Expressions.md).
- **Const Generics**: Type parameters with values `[T; N]`. See [Generics](./10_Generics.md).
- **Contracts**: Interfaces and traits. See [Contracts](./09_Contracts.md).
- **Control Flow**: `if`, `match`, loops. See [Control Flow](./06_Control_Flow.md).

## D
- **Defer**: Execution at scope exit. See [Control Flow](./06_Control_Flow.md).
- **Destructuring**: Extracting values from structs/tuples. See [Pattern Matching](./11_Pattern_Matching.md).

## E
- **Enums**: Algebraic data types. See [Enums](./08_Enums.md).
- **Error Handling**: `Result`, `Option`, `!>`. See [Error Handling](./17_Error_Handling.md).

## F
- **FFI**: Foreign Function Interface. See [Raw Pointers and FFI](./18_Raw_Pointers_and_FFI.md).
- **For Loop**: Iteration. See [Control Flow](./06_Control_Flow.md).
- **Functions**: Definitions and calls. See [Functions](./05_Functions_and_Methods.md).

## G
- **Generics**: Type parametrization. See [Generics](./10_Generics.md).
- **Goroutines**: Lightweight threads. See [Concurrency](./13_Concurrency.md).

## I
- **If/Elif/Else**: Conditional branching. See [Control Flow](./06_Control_Flow.md).
- **Impl**: Implementing contracts or methods. See [Contracts](./09_Contracts.md).
- **Imports**: Module system. See [Modules](./15_Modules_and_Imports.md).
- **Inline Assembly**: Direct CPU instructions. See [Inline Assembly](./29_Inline_Assembly.md).

## L
- **Let**: Variable declaration. See [Variables](./04_Variables_and_Constants.md).
- **Lifetimes**: Reference validity scopes. See [Memory Management](./14_Memory_Management.md).

## M
- **Match**: Pattern matching. See [Pattern Matching](./11_Pattern_Matching.md).
- **Memory Layout**: Struct alignment and padding. See [Structs](./07_Structs_and_Data_Types.md).
- **Methods**: Functions attached to types. See [Functions](./05_Functions_and_Methods.md).
- **Modules**: Code organization. See [Modules](./15_Modules_and_Imports.md).
- **Mutability**: `let!` and `&T!`. See [Variables](./04_Variables_and_Constants.md).

## O
- **Operators**: Symbolic operations. See [Operator Overloading](./23_Operator_Overloading.md).
- **Option**: Nullable value. See [Enums](./08_Enums.md).
- **Ownership**: Memory management rules. See [Memory Management](./14_Memory_Management.md).

## P
- **Package Manager**: `vex-pm`. See [Package Manager](./19_Package_Manager.md).
- **Pattern Matching**: Destructuring and matching. See [Pattern Matching](./11_Pattern_Matching.md).
- **Pointers**: Raw pointers `*T`. See [Raw Pointers](./18_Raw_Pointers_and_FFI.md).
- **Policies**: Metadata and attributes. See [Policy System](./20_Policy_System.md).

## R
- **Result**: Error handling type. See [Error Handling](./17_Error_Handling.md).
- **Return**: Exiting functions. See [Functions](./05_Functions_and_Methods.md).

## S
- **Standard Library**: Core functionality. See [Standard Library](./16_Standard_Library.md).
- **Structs**: Data structures. See [Structs](./07_Structs_and_Data_Types.md).

## T
- **Testing**: Built-in test framework. See [Testing](./26_Testing_Infrastructure.md).
- **Types**: Type system overview. See [Type System](./03_Type_System.md).

## U
- **Unsafe**: Unsafe operations. See [Raw Pointers](./18_Raw_Pointers_and_FFI.md).

## V
- **Variables**: `let` and constants. See [Variables](./04_Variables_and_Constants.md).
- **Vectorization**: SIMD/GPU support. See [Vectorization](./27_Vectorization_and_Autograd.md).
- **Visibility**: `public`, `private`, `readonly`. See [Structs](./07_Structs_and_Data_Types.md).

## W
- **While Loop**: Iteration. See [Control Flow](./06_Control_Flow.md).
